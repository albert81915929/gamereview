## Install

```sh
npm install
```

## Usage

```sh
to start server
  cd api
  npm start

to start client
  cd client
  npm start
```

## Run tests

```sh
to start server
  cd api
  DEBUG=gamereviewwebsite:* npm start

  or
  cd api
  nodemon /bin/www to enter interactive development mode

to start client
  cd client
  npm start
```
